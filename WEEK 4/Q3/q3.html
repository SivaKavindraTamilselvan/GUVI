Difference Between Document, Window, and Screen

Document, Window, and Screen in JavaScript/HTML
Feature Window Screen Document
Definition Represents the browser window that contains the web page. Represents the physical screen of the device. Represents the HTML document loaded in the browser.
Purpose Interacts with the browser window (size, location, alerts, etc.). Provides information about the screen's resolution, color depth, etc. Manipulates the content (HTML) of the web page, including elements and structure.
Common Uses Controlling window size, opening new windows, handling events (resize, scroll). Detecting screen size for responsive design and accessing screen properties. Modifying HTML elements (e.g., `document.getElementById`) and responding to user actions.
Access Accessed globally as `window` object. Accessed via `screen` object. Accessed via `document` object.
Example Code window.innerWidth
window.alert("Hello") 		screen.width
screen.height 		document.getElementById("elementId")
